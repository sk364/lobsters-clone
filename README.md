# Lobsters Clone

A clone of [Lobsters](https://lobste.rs/).

# Installation

* Install nvm and use latest node
* Clone the repo and `cd` into it.

```
>>> npm i
```

# Run Dev Server

```
>>> npm start
```

Should auto-open a new tab in the browser pointing to `http://localhost:3000`.

# Build

```
>>> npm run build
```
